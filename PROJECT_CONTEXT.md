# River2 --- мастер-контекст проекта

Дата: июль 2026

## Цель проекта

Создать современный премиальный сайт прогулок на катерах по
Санкт‑Петербургу с упором на: - высокую конверсию; - сильный визуальный
стиль; - SEO; - быструю работу на Next.js.

------------------------------------------------------------------------

# Технологии

-   React
-   TypeScript
-   Tailwind CSS

# Главный принцип разработки

Не переписывать проект с нуля.

Каждое изменение должно улучшать существующий код без потери
функциональности.

Работаем преимущественно с существующими файлами.
