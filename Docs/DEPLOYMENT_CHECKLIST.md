# Важно

До завершения разработки НЕ переносить сайт на основной домен.

Сначала проект полностью собирается и тестируется локально.

После завершения всех TASK выполняется только один финальный перенос на production.

Это позволит избежать простоев сайта и сохранить SEO существующего домена.
